# DataScience-Capstone-S&P500-Index-and-VWAC-ML
Contains datasets and code used in our Capstone project

Filename	Source	Description
DFF.csv	                      https://fred.stlouisfed.org/series/DFF	                Effective Federal Funds Rate
historical-cpi-u-202603.xlsx	https://www.bls.gov/cpi/tables/supplemental-files/	    Consumer Price Index
sp500_companies.csv	          https://www.kaggle.com/datasets/andrewmvd/sp-500-stocks	Contains company identification metrics.
sp500_index.csv	              https://www.kaggle.com/datasets/andrewmvd/sp-500-stocks	Contains daily index points.
sp500_stocks.csv	            https://www.kaggle.com/datasets/andrewmvd/sp-500-stocks	Too large to upload to GitHub repository. Contains adjusted close costs used to calculate Volume-Weighted Average Adjusted Close Cost per day.
T10Y2Y.csv	                  https://fred.stlouisfed.org/series/T10Y2Y	              10-Year minus 2-Year Treasury yield spread
T10YIE.csv	                  https://fred.stlouisfed.org/series/T10YIE	              10-Year Breakeven Inflation Rate
UNRATE.csv	                  https://fred.stlouisfed.org/series/UNRATE	              Unemployment Rate
VIXCLS.csv	                  https://fred.stlouisfed.org/series/VIXCLS	              CBOE VIX datasource for market regime classifications (high or low).
